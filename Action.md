Чтобы запустить логический план преобразований, мы выполняем действие (action).
Самое простое действие это `count()` возвращающее общее количество записей в DataFrame.

```scala
df.count()
```
Существует 3 вида действий:
- Действия для просмотра в консоли
- Действия для сбора данных в нативные объекты соответствующего языка
- Действия для записи данных в выходные источники
