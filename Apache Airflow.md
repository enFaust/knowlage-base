Концептуально Airflow состоит из трех основных компонентов:
- [[Планировщик Airflow]] - анализирует [[DAG]] и проверяет параметр schedule_interval. Если параметр в порядке планирует [[DAG]] для выполнения задач.
- [[Workers]] - Несут ответственность за фактическое выполнение работы. Они выбирают запланированные задачи и выполняют их.
- [[Веб-сервер Airflow]] - DAG визуализация, пользовательский UI для отслеживания графов и их разультатов.
