Это типобезопасная версия [[Неструктурированные API|структурированного API ]]. Он предназначен для написания статически типизированного кода на Java и Scala. Он не доступен для Python и R, так как эти языки динамически типизированные.

Dataset позволяет привязать Java/Scala класс к записям внутри [[DataFrame]] и работать с ними как с коллекцией типизированных объектов - похожей на `ArrayList` в Java или `Seq` в Python

Методы Dataset типобезопасны, `Dataset[Person]` гарантированно будет содержать объект Person. Класс `Dataset` параметризуется `Dataset<T>`  в Java и  `Dataset[T]` в Scala.
