- **Inner Join** - выводит только совпадающие записи по Join Condition.
	![[Pasted image 20251202111755.png]]
- **Outer Join** - выводит не только совпадающие записи, но и не совпадающие. 
	- **Left Join** - возвращает пересечение множеств и все элементы левого множества.
		![[Pasted image 20251202111914.png]]
	- **Right Join** - возвращает пересечение множеств и все элементы правого множества.
		![[Pasted image 20251202111925.png]]
	- **Full Join** - Возвращает обе таблицы объедененные в одну.
		![[Pasted image 20251202111933.png]]

- **Semi Join** -   Это по сути фильтр, чтобы найти пересекающиеся записи.
	- **leftsemi** - возвращает записи из левого множества, у которого есть соответвия в правом.
	- **rightsemi** - возвращает записи из правого множества, у которого есть соответвия в левом.
		![[Pasted image 20251202112832.png]]

- **Cross Join** - это декартово произведение множеств. Собирает все возможные пары из обоих таблиц. Каждый элемент одной таблицы объеденяется с каждым эллементом другой таблицы. 
		![[Pasted image 20251202112316.png]]

- **Self Join** - это соединение внутри одной таблицы. Join таблицы саму на себя.
